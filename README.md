# JavaScript Callbacks, Promises, and Async/Await Example

This project demonstrates how to use **callbacks**, **promises**, and **async/await** in JavaScript for fetching data from an API.

## Project Structure:
- **index.html**: The main HTML page to load the JavaScript files.
- **/js/fetchData.js**: Contains the functions that handle data fetching using callbacks, promises, and async/await.
- **/js/main.js**: Calls the functions and logs the responses in the console.
- **/styles/style.css**: Optional styles for the page.

## How to run:
1. Clone or download this project.
2. Open `index.html` in a web browser.
3. Open the browser console to see the logged results from the JavaScript functions.

## Example Data:
This project fetches data from a placeholder API: https://jsonplaceholder.typicode.com/posts/1
